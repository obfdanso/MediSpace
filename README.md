This is a React + Vite web app.

## Getting Started

First, run the development server:

```bash
npm install
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open the local URL printed in your terminal (usually `http://localhost:3000`) with your browser to see the result.

You can start editing pages in `src/pages/`. The app auto-updates as you edit files.

Fonts are loaded via `@fontsource/*` in `app/globals.css`.

## Build and preview

```bash
npm run build
npm run preview
```
